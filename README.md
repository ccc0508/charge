# 💰 记账本 - 微信小程序

一款简洁易用的个人记账微信小程序，帮助你轻松管理日常收支。

## ✨ 功能特性

- 📝 **快速记账** — 支持收入/支出记录，含金额、分类、备注、日期
- 📊 **数据统计** — 饼图可视化收支构成，按月查看收支概况
- 🏆 **排行榜** — 支出/收入 TOP 排行，一目了然
- 📅 **月度导航** — 按月切换，查看历史账单
- 👆 **左滑删除** — 流畅的手势交互，快速删除记录
- 📱 **下拉刷新** — 随时刷新最新数据

## 📸 页面概览

| 页面 | 说明 |
|------|------|
| 首页 (`index`) | 月度汇总 + 按日分组的账单列表 |
| 记账 (`add`) | 新增收入/支出记录 |
| 统计 (`stats`) | 饼图构成 + 分类图例 + 排行榜 |

## 🏗️ 项目结构

```
charge/
├── app.js                  # 小程序入口
├── app.json                # 全局配置
├── app.wxss                # 全局样式
├── pages/
│   ├── index/              # 首页 - 账单列表
│   ├── add/                # 记账页 - 新增记录
│   ├── stats/              # 统计页 - 图表分析
│   └── logs/               # 日志页
├── utils/
│   ├── models.js           # 数据模型 & 分类常量
│   ├── storage.js          # 本地存储操作
│   ├── store.js            # 数据聚合 & 统计计算
│   └── util.js             # 工具函数
├── project.config.json     # 项目配置
└── sitemap.json            # 小程序索引配置
```

## 🔧 技术栈

- **框架**: 微信小程序原生框架
- **语言**: JavaScript / WXML / WXSS
- **数据存储**: `wx.setStorageSync` / `wx.getStorageSync` (本地存储)
- **图表绘制**: Canvas 2D (饼图)

## 🚀 快速开始

1. 克隆仓库
   ```bash
   git clone https://github.com/ccc0508/charge.git
   ```
2. 打开 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
3. 导入项目，选择克隆下来的 `charge` 文件夹
4. 点击编译即可预览
